Usage

A simple example which add both reqValue and intValue validation to the form named myform2 and field req_field2. It uses showAppend message type with a fade effect:

var b = new daValidator("myform2",{msgType : "showAppend",msgEffect : "fade"},$);
b.addValidation("reqValue","req_field2","REQUIRED field");
b.addValidation("intValue","int_field2","INTEGER field");

Message types:

  • showAppend
  • showDialog
  • showToolTip
  • showAlert

Validation types:

  • reqValue
  • intValue
  • urlValue
  • minLength
  • maxLenght
  • emailValue
  • regExprValue
